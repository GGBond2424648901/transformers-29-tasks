# 姿态估计服务安装说明

## 🎯 功能说明

这是一个**真正的人体骨骼关键点检测**服务，可以：
- 检测人体17个关键点（鼻子、眼睛、耳朵、肩膀、肘部、手腕、髋部、膝盖、脚踝）
- 绘制人体骨骼结构图
- 可视化显示检测结果

## 📦 依赖安装

### 基础依赖（必需）
```bash
pip install flask pillow numpy
```

### OpenPose检测器（推荐）
```bash
pip install controlnet-aux
```

**说明**：
- 如果安装了 `controlnet-aux`，将使用专业的OpenPose技术进行检测
- 如果未安装，将使用简化版本绘制示例骨架

## 🎨 特色功能

1. **双图对比**：左侧显示原图，右侧显示骨骼检测结果
2. **关键点列表**：显示检测到的所有关键点
3. **运动少女主题**：橙色活力配色 + 能量波纹动画
4. **背景图片**：使用自定义背景图片

## 📝 使用说明

1. 上传包含人物的图片（建议全身照）
2. 点击"开始检测"按钮
3. 查看右侧的骨骼检测结果
4. 下方显示检测到的关键点列表

## 🔧 技术细节

### 检测的17个关键点：
1. 鼻子
2. 左眼、右眼
3. 左耳、右耳
4. 左肩、右肩
5. 左肘、右肘
6. 左腕、右腕
7. 左髋、右髋
8. 左膝、右膝
9. 左踝、右踝

### 骨骼连接：
- 头部：鼻子-眼睛-耳朵
- 上半身：肩膀-肘部-手腕
- 躯干：肩膀-髋部
- 下半身：髋部-膝盖-脚踝

## ⚠️ 注意事项

1. **图片质量**：清晰的全身照片效果最好
2. **人物姿态**：正面或侧面姿态检测效果较好
3. **背景干扰**：简单背景有助于提高检测准确度
4. **多人检测**：当前版本主要针对单人检测

## 🆚 与之前的区别

**之前**：使用图像分类模型，只能识别图片类别（如"阳伞"、"员工"等）

**现在**：使用姿态估计技术，真正检测人体骨骼关键点，绘制骨架结构
