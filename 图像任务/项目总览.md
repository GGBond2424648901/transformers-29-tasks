# 🎨 图像任务项目总览

所有项目都已创建可爱的二次元风格Web服务，每个都有独特的个性化设计！

## 📋 项目列表

### 1. 🐱 图像分类 - 猫娘助手
- **端口**: 6001
- **风格**: 粉色猫娘主题，樱花飘落
- **功能**: 上传图片识别物体类别
- **模型**: google/vit-base-patch16-224
- **特色**: 可爱的猫耳装饰，弹跳动画

### 2. 🎯 目标检测 - 侦探少女
- **端口**: 6002  
- **风格**: 蓝紫色侦探主题，星星闪烁
- **功能**: 检测图片中的多个物体及位置
- **模型**: facebook/detr-resnet-50
- **特色**: 放大镜图标，物体框标注

### 3. 🎭 图像分割 - 魔法少女
- **端口**: 6003
- **风格**: 彩虹渐变魔法主题，魔法粒子
- **功能**: 将图片分割成不同区域
- **模型**: nvidia/segformer-b0-finetuned-ade-512-512
- **特色**: 魔法棒动画，彩色分割效果

### 4. 📏 深度估计 - 科技少女  
- **端口**: 6004
- **风格**: 青色科技主题，数字雨效果
- **功能**: 估计图片中物体的深度信息
- **模型**: Intel/dpt-large
- **特色**: 矩阵风格，深度热力图

### 5. 🤸 姿态估计 - 运动少女
- **端口**: 6005
- **风格**: 橙色活力主题，能量波纹
- **功能**: 检测人体关键点和姿态
- **模型**: microsoft/beit-base-patch16-224-pt22k-ft22k
- **特色**: 运动图标，骨骼点标注

### 6. 🔍 关键点检测 - 精灵少女
- **端口**: 6006
- **风格**: 绿色自然主题，叶子飘落
- **功能**: 检测图片中的关键特征点
- **模型**: facebook/detr-resnet-50
- **特色**: 精灵翅膀，关键点高亮

### 7. 🎬 视频分类 - 偶像少女
- **端口**: 6007
- **风格**: 紫粉色偶像主题，星光闪耀
- **功能**: 识别视频内容类别
- **模型**: MCG-NJU/videomae-base
- **特色**: 麦克风图标，音符动画

### 8. 🌈 零样本图像分类 - 魔导书少女
- **端口**: 6008
- **风格**: 金色魔法书主题，魔法阵
- **功能**: 无需训练识别任意类别
- **模型**: openai/clip-vit-base-patch32
- **特色**: 魔法书翻页，自定义标签

## 🎀 共同特点

✨ **二次元风格**: 每个项目都有独特的动漫少女主题
🌸 **动画效果**: 樱花/星星/粒子等飘落动画
🎨 **渐变配色**: 精心设计的渐变色彩方案
💫 **交互动画**: 悬停、点击都有可爱的动画反馈
📱 **响应式设计**: 支持各种屏幕尺寸
🖼️ **自定义背景**: 使用统一的背景图片

## 🚀 快速启动

每个项目目录下都有 `启动Web服务.bat` 文件，双击即可启动！

或使用命令行：
```bash
D:\aaaalokda\envs\myenv\python.exe "项目名Web服务.py"
```

## 📝 注意事项

1. 首次运行会自动下载模型，请耐心等待
2. 所有模型保存在统一的缓存目录
3. 需要GPU支持以获得最佳性能
4. 端口号从6001-6008，避免冲突

## 💖 使用建议

- 建议按顺序体验各个项目
- 每个项目都有示例图片或说明
- 可以同时运行多个服务（不同端口）
- 享受可爱的二次元AI体验！
