## 🎨 设计特点

### 主题配色方案
- **猫娘助手**: 粉色系 (#ffb6c1, #ff69b4)
- **侦探少女**: 蓝紫色系 (#4169e1, #6a5acd)

- **魔导书少女**: 金色系 (#ffd700, #ff8c00)

### 动画效果
所有项目都实现了：
1. **持续飘落动画**: 使用JavaScript setInterval每200-350ms创建新粒子
2. **动态旋转/移动**: 粒子从顶部飘落到底部，带有旋转效果
3. **渐变透明度**: 粒子在飘落过程中逐渐变淡
4. **交互反馈**: 按钮悬停、点击都有流畅的动画效果
5. **图标动画**: 顶部图标有浮动、旋转、跳动等特效

### 技术实现
- **前端**: HTML5 + CSS3 + JavaScript
- **后端**: Flask + Transformers
- **动画**: CSS @keyframes + JavaScript setInterval
- **响应式**: 支持各种屏幕尺寸
- **GPU加速**: 所有模型都使用CUDA加速

## 💡 使用说明

### 通用步骤
1. 启动对应的Web服务
2. 在浏览器中打开访问地址
3. 点击上传区域或拖拽图片
4. 点击"开始XXX"按钮
5. 查看识别结果

### 特殊说明
- **零样本图像分类**: 可以自定义识别类别，无需训练
- **视频分类**: 目前支持单帧图片分类
- **深度估计**: 会生成深度热力图
- **目标检测/关键点检测**: 会在图片上标注检测框

## 📝 注意事项

1. **首次运行**: 会自动下载模型，请耐心等待（几百MB到几GB不等）
2. **模型缓存**: 所有模型保存在 `D:\transformers训练\transformers-main\预训练模型下载处`
3. **GPU内存**: 建议至少4GB显存，8GB更佳
4. **端口占用**: 确保6001-6008端口未被占用
5. **浏览器兼容**: 推荐使用Chrome、Edge等现代浏览器
