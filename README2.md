## 🔧 常见问题

### Q1: 如何选择合适的任务？

**A**: 根据你的需求和兴趣：
- 文本处理 → 文本任务（问答、分类、翻译等）
- 图像处理 → 图像任务（分类、检测、分割等）
- 语音处理 → 音频任务（识别、合成等）
- 综合应用 → 多模态任务（图文结合等）

### Q2: Web服务启动失败怎么办？

**A**: 检查以下几点：
1. 端口是否被占用（可以修改代码中的端口号）
2. 依赖是否安装完整（pip install flask transformers torch）
3. 模型是否下载完成（首次运行需要下载模型）
4. Python环境是否正确

### Q3: 需要GPU吗？

**A**: 
- 小模型（< 500M）：CPU即可，但GPU更快
- 大模型（> 1GB）：强烈建议使用GPU
- 实时任务（摄像头）：必须使用GPU

本项目已配置CUDA 12.1支持。

### Q4: 模型下载慢怎么办？

**A**: 
- 模型会自动下载到：`D:\transformers训练\transformers-main\预训练模型下载处`
- 首次运行需要等待下载完成
- 可以使用镜像源加速
- 也可以手动下载模型文件

### Q5: 如何修改Web服务的UI？

**A**: 
- 每个Web服务都是独立的Python文件
- HTML/CSS/JavaScript都在文件中
- 可以直接修改样式、颜色、布局等
- 修改后重启服务即可看到效果

### Q6: 如何获取帮助？

**A**: 
1. 查看任务文件夹中的README.md
2. 参考官方文档：https://huggingface.co/docs/transformers
3. 查看示例代码中的注释
4. 查看`项目完成总结.md`了解详细信息
