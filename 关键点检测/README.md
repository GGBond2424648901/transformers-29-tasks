# 🔍 关键点检测与匹配（Keypoint Detection & Matching）

## 📖 任务简介
检测图像中的特征点并在不同图像间进行匹配，用于图像配准、3D重建等任务。

## 🎯 应用场景
- 📷 图像拼接、全景图生成
- 🏗️ 3D 重建、建筑测量
- 🤖 SLAM、机器人导航
- 🔍 物体识别与追踪

## 🛠️ 推荐算法
- **SuperPoint + SuperGlue**: 深度学习方法，效果最好
- **SIFT**: 经典算法，鲁棒性强
- **ORB**: 快速，适合实时应用

## 💡 快速开始
```bash
pip install opencv-python
python 关键点检测示例.py
```
