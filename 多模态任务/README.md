# 🌐 多模态任务（Multimodal Tasks）

## 📖 简介

多模态任务结合了多种类型的数据（图像、文本、音频等），实现更强大的 AI 功能。

## 📁 任务列表

### 1. 视觉问答（VQA）
根据图像回答问题
- **模型**: BLIP, LLaVA
- **应用**: 智能客服、图片搜索

### 2. 图像描述生成
自动为图像生成文字描述
- **模型**: BLIP-2, ViT-GPT2
- **应用**: 无障碍辅助、图片 SEO

### 3. 文档理解
OCR + 文档结构理解
- **模型**: LayoutLM, Donut
- **应用**: 发票识别、合同分析

### 4. 表格问答
对结构化表格数据进行问答
- **模型**: TAPAS, TAPEX
- **应用**: 数据分析、财务报表

### 5. 音频+文本理解
多模态音频内容理解
- **模型**: Qwen2-Audio
- **应用**: 会议纪要、客服分析

### 6. 视觉+文本生成
基于图像和文本的生成
- **模型**: LLaVA, Qwen-VL
- **应用**: AI 助手、图像编辑

## 🚀 快速开始

每个子文件夹都包含：
- 示例代码（.py）
- 使用说明（README.md）

```bash
cd 实战训练/多模态任务/视觉问答
python 视觉问答示例.py
```

## 💡 学习路径

1. **入门**: 视觉问答、图像描述生成
2. **进阶**: 文档理解、表格问答
3. **高级**: 音频文本理解、视觉文本生成
