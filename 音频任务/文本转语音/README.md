# 文本转语音 (Text-to-Speech)

## 📝 项目简介

使用 `facebook/mms-tts-chz` 模型将文本转换为语音，支持中文。

## 🎯 功能特点

- ✅ 支持中文文本转语音
- ✅ 支持英文文本转语音
- 实时生成音频
- 可下载生成的音频文件

## 🚀 快速开始

### 运行Web服务

```bash
python 文本转语音Web服务.py
```

服务将在 `http://localhost:9005` 启动

### 运行示例代码

```bash
python 文本转语音示例.py
```

## 📊 模型信息

- **模型**: `facebook/mms-tts-chz`
- **任务**: Text-to-Speech (文本转语音)
- **语言支持**: ✅ 完全支持中文和英文

## ⚠️ 注意事项

### 语言支持
- ✅ **中文**: 完全支持
- ✅ **英文**: 完全支持

### 使用建议
1. 文本长度建议在200字以内
2. 首次使用需要下载模型（约145MB）
3. 使用HF镜像加速下载

## 🎨 Web界面特点

- 动漫风格UI设计
- 樱花飘落动画效果
- 实时音频预览
- 一键下载功能

## 🔧 技术栈

- Flask Web框架
- Transformers库
- Facebook MMS-TTS模型
- SciPy音频处理

## 📌 端口信息

- Web服务端口: 9005

## 🌟 相关服务

- 语音识别 (端口 9003) - 支持中文
- 语音到语音 (端口 9002) - 识别支持中文,合成仅英文
- 音频分类 (端口 9004) - 语言无关

## 🔄 模型下载

如需手动下载模型：

```bash
python 下载中文模型.py
```

模型会自动下载到配置的缓存目录。
